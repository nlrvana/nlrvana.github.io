先看第一处  

这个过滤大概意思就是删除了`__builtins__`，但是保留了一些函数  
第二处  
```python  
    assert 0 &lt; len(code) &lt; 666, &#34;Boy your code is too long!!!!!&#34;  
    assert all(i not in code for i in list(&#34;&#39;\&#34;_{}#&#34;)), &#34;I don&#39;t like those chars...&#34;
```  
